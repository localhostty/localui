# Collapsible Primitive

<div style="">
  <div style="border: 2px solid black; padding: 12px; height: 112px; display: flex; align-items: center; justify-content: center">

      ```
      <Collapsible.Root>

        </Collapsible.Trigger>
        <Collapsible.Panel
          keepMounted={true}
          style="width: 250px; border: 2px solid black; padding: 4px; margin-top: 4px;"
      ```

## API

### Collapsible.Root

<table>
  <tbody style="border: 2px solid black; padding: 2px;">
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        defaultOpen?: boolean (default: false)
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Default State of the Collapsible
      </td>
    </tr>
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        open?: boolean (default: false)
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Current State of the Collapsible
      </td>
    </tr>
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        onOpenChange?: Setter(boolean)
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Method to toggle open state
      </td>
    </tr>
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        disabled?: boolean (default: false)
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Whether to disable the collapsible element.
      </td>
    </tr>
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        forwardedRef?: HTMLDivElement
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Ref to the root div element
      </td>
    </tr>
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        internal_id?: string (default: generated automatically)
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Identifier of the CollapsiblePanel (should be unique). This is usually
        internally generated and we advise to not modify it unless 100%
        necessary.
      </td>
    </tr>
  </tbody>
</table>

### Collapsible.Trigger

<table>
  <tbody style="border: 2px solid black; padding: 2px;">
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        forwardedRef?: HTMLButtonElement | HTMLDivElement
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Ref to the root button/div element
      </td>
    </tr>
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        nativeButton?: boolean (default: true)
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Whether to use a semantic HTML &lt;button&gt; Element or a &lt;div&gt;
        with type button.
      </td>
    </tr>
  </tbody>
</table>

### Collapsible.Panel

<table>
  <tbody style="border: 2px solid black; padding: 2px;">
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        forwardedRef?: HTMLDivElement
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Ref to the root div element
      </td>
    </tr>
    <tr style="border: 2px solid black; padding: 2px;">
      <td style="border: 2px solid black; padding: 2px;">
        keepMounted?: boolean (default: false)
      </td>
      <td style="border: 2px solid black; padding: 2px;">
        Whether to remove the Element from the DOM entirely or just hide it.
      </td>
    </tr>
  </tbody>
</table>
