# localui

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

UI Component Library for SolidJS

## Quick start

Install it:

```bash
npm i localui
# or
yarn add localui
# or
pnpm add localui
# or
bun add localui
```
