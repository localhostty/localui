# localui

[![bun](https://img.shields.io/badge/maintained%20with-bun-cc00ff.svg?style=for-the-badge&logo=bun)](https://bun.com/)

UI Component Library for SolidJS

## Quick start

Install it:

```bash
npm i localui
# or
yarn add localui
# or
pnpm add localui
# or
bun add localui
```

Use it:

```tsx
import localui from 'localui'
```
